# Security Policy

Security is critical to UTOID. We welcome responsible disclosure of
vulnerabilities and work with reporters to coordinate fixes.

## Where to report

Use GitHub Security Advisories in the affected repository whenever possible.
This supports private disclosure and coordinated patching.

If Security Advisories are not available, contact the repository owners via
GitHub profiles and share the details privately.

## What to include

Please provide:

- Affected repository and commit or version
- Reproduction steps or proof of concept
- Impact assessment and potential exploitability
- Any suggested mitigations or patches

## Scope

Security reports should target the UTOID repositories, including utocoin core,
ipv7, utowallet, and this entry repository.

## Coordinated disclosure

We aim to acknowledge reports as soon as practical and work with reporters on a
coordinated disclosure timeline. Please avoid public disclosure until a fix or
mitigation is available.
