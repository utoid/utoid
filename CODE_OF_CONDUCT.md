# Code of Conduct

We are committed to providing a welcoming, inclusive, and professional
community for everyone participating in the UTOID ecosystem.

## Our standards

Examples of behavior that contributes to a positive environment:

- Using inclusive and respectful language
- Focusing on technical merit and evidence-based discussion
- Accepting constructive feedback and different viewpoints
- Showing empathy toward other community members

Examples of unacceptable behavior:

- Harassment, discrimination, or personal attacks
- Trolling, doxxing, or publishing private information
- Persistent disruption of discussions or development work

## Scope

This Code of Conduct applies within project spaces and in public spaces when an
individual's behavior represents the project or its community. This includes
GitHub repositories, issues, pull requests, and official communication
channels.

## Enforcement

If you experience or witness unacceptable behavior, please contact the
repository owners via GitHub profiles. If no private channel is available, you
may open an issue clearly labeled "conduct" and a maintainer will follow up.

Project maintainers may take any action they deem appropriate, including
warning, temporary bans, or permanent removal from community spaces.

## Acknowledgement

This policy is inspired by common open-source community standards and adapted
for the UTOID project.
